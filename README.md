📚 Library Management System

A full-stack Library Management System designed to streamline operations for both students and teachers. This project provides a complete ecosystem to manage books, users, and transactions like issuing and returning books efficiently.


🚀 Project Overview

This system allows libraries to:

Manage book inventory
Handle student and admin access
Track issued and returned books
Calculate and display fines

It ensures smooth coordination between admin (teacher) and students, with proper rules like borrowing limits and fine tracking.

🛠️ Tech Stack
Java (JSP/Servlets)
MySQL
NetBeans IDE
Apache Tomcat (Server)


✨ Features

🔐 Admin Login & Student Login
➕ Add and Manage Books
📖 Issue Books (Max 3 per user)
🔄 Return Books
🔍 Search Books
💰 Fine Calculation & Display
👤 User Management


⚙️ How It Works

Admin logs into the system
Admin can add/manage books and users
Students log in to:
Search books
Issue books (up to 3 limit)
Return books
System tracks issued books and calculates fines
Students can view their issued books and fines

🧪 How to Run the Project

✅ Using NetBeans (Recommended)
Clone or download the repository

Select this project folder
Configure Apache Tomcat Server
Right-click project → Run
